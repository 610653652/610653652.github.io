1.寻找出现奇数次的一个数

在一个数组中，有一个数出现了奇数次，其他数都出现了偶数次，如何找出出现奇数次的数。

令eor=0，令eor和数组中全部数异或（^）一遍，eor等于几，那个数就是几

public static void printOddTimesNum1(int[] arr){
    int eor = 0;
    for(int i = 0;i<arr.length;i++){
        eor = eor ^ arr[i];
    }
    System.out.println(eor);
}

2.寻找出现奇数次的两个数

在一个数组中，有两个数出现了奇数次，其他数都出现了偶数次，如何找出出现奇数次的两个数。

public static void printOddTimesNum2(int[] arr){
    int eor = 0;
    for (int curNum : arr){
        eor ^= curNum;
    }
    //eor = a ^ b
    //eor != 0
    //eor二进制在某一个位置必定为1
    int rightOne = eor & (~eor + 1); //提取最右侧的1
    
    int eor1 = 0;
    for(int cur : arr){
        if((cur & rightOne) == 0){ //只和rightOne位为1的进行异或,找出a或b其中一个值
            eor1 ^= cur; //eor1是a或b其中的一个
        }
    }
    System.out.println(eor1 +' '+ (eor1^eor));
}

public static void SelectionSort(int[] arr) {
        if (arr == null || arr.length < 2) {
            return;
        }
        for (int i = 0; i < arr.length; i++) {
            int minIndex = i;
            for (int j = i + 1; j < arr.length - 1; j++) {
                minIndex = arr[j] < arr[minIndex] ? j : minIndex;
            }
            swap(arr, i, minIndex);
        }
    }

    if(L==R){	//arr[L..R]范围上只有一个数,直接返回
        return arr[L];
    }
    int mid = L +((R-L) >> 1); //中点
    int leftMax = process(arr,L,mid);
    int rightMax = process(arr,mid+1,R);
    return Math.max(leftMax,rightMax);
}

public static void process(int[] arr,int L,int R){
    if(L==R){	//arr[L..R]范围上只有一个数,直接返回
        return arr[L];
    }
    int mid = L +((R-L) >> 1); //中点
    process(arr,L,mid);
    process(arr,mid+1,R);
    merge(arr,L,mid,R);
}

public static void merge(int[] arr,int L,int M,int R){
    int[] help = new int[R-L+1];
    int i = 0;
    int p1 = L;
    int p2 = M+1;
    while(p1<=M && p2<=R){	//都不越界的情况下
        help[i++] = arr[p1] <= arr[p2] ? arr[p1++] : arr[p2++];	
        //比较p1位置的值和p2位置的值,谁小谁拷贝到help[]的i位置,然后被拷贝那侧指针右移
    }
    //一侧指针到头以后,将另一侧指针剩下的数直接拷贝到help[]中
    while(p1<=M){
        help[i++] = arr[p1++];
    }
    while(p2<=R){
        help[i++] = arr[p2++];
    }
    //将help[]中的东西拷贝回arr中
    for (i = 0; i < help.length; i++){
        arr[L + i] = help[i];
    }
}

9.小和问题

在一个数组中，每一个数左边比当前数小的数累加起来，叫做这个数组的小和。求一个数组的小和。

例子：[1,3,4,2,5] 1左边比1小的数，没有；3左边比3小的数，1；4左边比4小的数，1、3；2左边比2小的数，1；5左边比5小的数，1、3、4、2；所以小和为1+1+3+1+1+3+4+2=16

public static int smallSum(int arr[]){
    if (arr == null || arr.length < 2) {
            return;
        }
    return process(arr,0,arr.length-1);
}
//arr[L..R]既要排好序,也要求小和
public static int process(int[] arr, int l, int r){
    if(l==r){
        return 0;
    }
    int mid = L +((R-L) >> 1); //中点
    return process(arr,l,mid) + process(arr,mid+1,r) + merge(arr,l,mid,r); 
} 
public static int merge(int[] arr,int l,int m,int r){
    int[] help = new int[r-l+1];
    int i = 0;
    int p1 = l;
    int p2 = m+1;
    int res = 0;
    while(p1<=m && p2 <= r){
        //当左侧指针(p1)位置值小于右侧指针位置时,增加当前右组比p1大的数量(r - p2 + 1)的小和
        res += arr[p1] < arr[p2] ? (r - p2 + 1) * arr[p1] : 0;
        //左组比右组小的时候拷贝左组,否则拷贝右组(等于时先拷贝右组)
        help[i++] = arr[p1] < arr[p2] ? arr[p1++] : arr[p2++];
    }
    while(p1<=m){
        help[i++] = arr[p1++];
    }
    while(p2<=r){
        help[i++] = arr[p2++];
    }
    for (i = 0; i < help.length; i++){
        arr[l + i] = help[i];
    }
    return res;
}

12.堆（理解为一种特殊的完全二叉树）

  • 完全二叉树

    从位置0开始的连续一段数组理解为完全二叉树

    索引 i 位置左孩子：2 * i + 1，右孩子：2 * i + 2，父节点：(i - 1) / 2

  • 大根堆，小根堆

    • 大根堆：以i节点为头的整棵树，节点i是最大值（每个子树的最大值是头节点的值）
    • 小根堆：以i节点为头的整棵树，节点i是最小值（每个子树的最小值是头节点的值）

  • heapInsert：将输入的节点排成大根堆

    public static void heapInsert(int[] arr, int index){
    while(arr[index] > arr[(index-1)/2]) {//和父节点比较大小
    	swap(arr,index,(index-1)/2);
        index = (index - 1)/2;
    }
}

  • heapify：把大根堆中最大的数拿掉，并让剩余的数变成大根堆

先将0位置上的数用临时变量记住，将最后一个数拷贝到0位置，heapSize--，然后从头节点开始，在其左孩子和右孩子中找到最大的，头节点和其比较大小。若头节点小于最大的孩子，则孩子变到头节点位置，头节点到该孩子位置。直到两个孩子的最大值都没有头节点大或头节点没有孩子时停止

public static void heapify(int[] arr, int index,int heapSize){
    int left = index * 2 + 1; //左孩子的下标
    int right = left + 1;
    while(left < heapSize){ //如果左孩子没有越界(判断下方有没有孩子)
        //如果右孩子没有越界且右比左大,则把下标给largest(寻找两个孩子中的最大值)
        int largest = right < heapSize && arr[right] > arr[left] ? right : left;
        //比较父节点和较大孩子谁大
        largest = arr[index] > arr[largest] ? index : largest;
        if(largest == index){
            break;
        }
        swap(arr,largest,index); //将父节点交换到最大(在孩子和当前父节点中最大)
        index = largest; //接着往下走
        left = index * 2 + 1;
    }
}

  • 将i位置的值a变成b，让新的数组重新变成大根堆

    判断b和i位置原来值(a)的大小，如果a>b（换来的数变小了），则向下进行heapify（把孩子中最大的数顶到替换位置）；如果a<b（变大了），则向上进行heapInsert（将这个变大后的数和父节点比较）

    如果不告诉b的大小，则先进行heapify，再进行heapInsert（只能中一个，先做谁无所谓）

  • 堆排序（时间复杂度O(N * logN)，空间复杂度O(1)）

    先用heapInsert把数组排成大根堆，然后将0位置和最后一位交换，heapsize--（最大值断开联系，排到正确位置），做heapify；重复将0位置和最后一位交换，heapsize--，直到heapsize=0

public class HeapSort{
    public static void heapSort(int[] arr){
        if (arr == null || arr.length < 2) {
            return;
        }
        //排成大根堆
        for(int i = 0; i < arr.length; i++){
            heapInsert(arr,i);
        }
        int heapSize = arr.length;
        //将0位置和最后一位交换，heapsize--
        swap(arr,0,--heapSize);
        //重复heapify和交换直到heapsize=0
        while(heapSize > 0){
            heapify(arr,0,heapSize);
            swap(arr,0,--heapSize);
        }
    }
    
}

13.堆排序扩展

已知一个几乎有序的数组，几乎有序是指，如果把数组排好顺序的话，每个元素移动的距离可以不超过k，并且k相对于数组来说比较小。请选择一个合适的排序算法针对这个数据进行排序。

public class SortArrayDistanceLessK{
    public void sortedArrDistanceLessK(int[] arr,int k){
        //生成小根堆
        PriorityQueue<Integer> heap = new PriorityQueue<>();
        int index = 0;
        //把前k个数放到小根堆中
        for(;index <= Math.min(arr.length,k);index++){
            heap.add(arr[index]);
        }
        //再加一个数,此时heap中有k+1个数,将第一个数弹出,再次添加
        int i = 0;
        for(;index <= arr.length;i++,index++){
            heap.add(arr[index]);
            arr[i] = heap.poll();
        }
        //当数组中数字加完时(heap中仍有剩余),将heap按顺序弹出
        while(!heap.isEmpty()){
            arr[i++] = heap.poll();
        }
    }
}

	public static int maxbits(int[] arr) {
        int max = Integer.MIN_VALUE;
        for (int i = 0; i < arr.length; i++) {
            max = Math.max(max,arr[i]);
        }
		int res = 0;

        }
		return res;
    }
    //arr[begin..end]排序,digit表示最大的数有几个十进制位
    public static void radixSort(int[] arr,int L,int R,int digit) {
        final int radix = 10;
        int i=0, j=0;
        //有多少个数准备多少个辅助空间
        int[] bucket = new int[R - L + 1];
        
        for (int d = 1; d <= digit; d++){ //有多少位就进出几次(发生多少次入桶出桶)
           //10个空间
           int[] count = new int[radix]; //count[0..9]
            //统计词频,每个位置的词频为≤这个位置数字的数量多少
            //count[0] 当前位(d位)是0的数字有多少个
           //count[1] 当前位(d位)是(0和1)的数字有多少个
           //count[2] 当前位(d位)是(0、1和2)的数字有多少个
           //count[i] 当前位(d位)是(0~i)的数字有多少个
            for(i = L;i <= R;i++){
                j= getDigit(arr[i],d);
                count[j]++;
            }
            //统计≤的数量
            for(i = 1; i < radix ;i++){
                count[i] = count[i] + count[i-1];
            }
            //分片,把第digit位的数按照之前统计的词频分片
            //比如个位≤3的数共有16个,则从右向左遍历第一次遇见个位为3的数(比如63),将其放到15号位
            for (i = R; i>= L;i--) {
                j = getDigit(arr[i], d);
                bucket[count[j] - 1] = arr[i];
                count[j]--;
            }
            //出桶,放到数组中
			for (i = L, j= 0; i<= R; i++, j++) {
                arr[i] = bucket[j];
        } 
    }
	public static int getDigit(int x, int d) {
        return ((x / ((int) Math.pow(10, d - 1))) % 10);
    }
}

21.判断一个链表是否为回文结构

  • 题目：给定一个单链表的头节点head，请判断该链表是否为回文结构。

  • 例子：1->2->1，返回true；1->2->2->1，返回true；15->6->15，返回true；1->2->3，返回false。

  • 要求：如果链表长度为N，时间复杂度达到0(N)，额外空间复杂度达到O(1)。

笔试：放到栈中，弹出以后和原来是否一样

优化：把右侧的东西放到栈里，栈弹出的内容和左面进行比对。（快慢指针：快指针走2，慢指针走1，快指针走完了，慢指针就会到中点了）（coding练熟）

面试：(1)先快慢指针，中点遍历时(2)将中点指空，(3)右侧往回指。左右两侧用引用标记，(4)两个标记往中间走，并进行比对。直到走到空，(5)如果每一步都一样，则是回文。返回之前需要把(6)右侧恢复原来的样子。

public class reverseList{
    //造链表
    public static class Node{
        public int value;
        public Node next;

        public Node(int data){
            this.value = data;
        }
    }
//笔试
    public static boolean isPalindrome1(Node head){
        Stack<Node> stack = new Stack<Node>(); //栈
        Node cur = head;    //指针
        while (cur != null){    //压栈
            stack.push(cur);
            cur = cur.next;
        }

        while (head != null){   //弹栈,逐个对比
            if(head.value != stack.pop().value){
                return false;
            }
            head = head.next;
        }
        return true;
    }
//面试
    public static boolean isPalindrome3(Node head){
        if (head == null || head.next == null){
            return true;
        }
        Node n1 = head;//慢指针
        Node n2 = head;//快指针
        while (n2.next != null && n2.next.next != null){    //(1)快慢指针寻找中间节点
            n1 = n1.next;
            n2 = n2.next.next;
        }
        n2 = n1.next;   //n2指向右侧第一个节点
        n1.next = null; //(2)中节点指向null
        Node n3 = null;
        while (n2 != null){ //(3)反转右半部分
            n3 = n2.next;
            n2.next = n1;
            n1 = n2;
            n2 = n3;
        }
        n3 = n1;
        n2 = head;
        boolean res = true;
        //(4)往中间节点走,(5)如果每一步都一样,则是回文。
        while(n1 != null && n2 != null){
            if (n1.value != n2.value){
                res = false;
                break;
            }
            n1 = n1.next;
            n2 = n2.next;
        }
        //(6)恢复右侧
        n1 = n3.next;
        n3.next = null;
        while (n1 != null){
            n2 = n1.next;
            n1.next = n3;
            n3 = n1;
            n1 = n2;
        }
        return res;
    }
}

    • 然后判断end1和end2地址是否一致，不一致则没有相交，一致则长链表先走差值步，然后短链表跟着走，最后两个链表一定会在第一个相交节点处相遇

    • loop1继续走，如果在转回自己的过程中：

public class FindFirstIntersectNode {
    public static class Node {
        public int value;
        public Node next;

        public Node(int data) {
            this.value = data;
        }
    }
    //判断是否有环,并返回入环的第一个节点
    public static Node getLoopNode(Node head){
        if(head == null || head.next == null || head.next.next ==  null){
            return  null;
        }
        Node n1 = head.next;    //慢指针
        Node n2 = head.next.next;   //快指针
        while(n1 != n2){
            //无环
            if(n2.next == null || n2.next.next == null){
                return null;
            }
            n2 = n2.next.next;
            n1 = n1.next;
        }
        n2 = head;
        while (n1 != n2){
            n1 = n1.next;
            n2 = n2.next;
        }
        return n1;
    }
    
    //两个单链表都是无环的情况
    public static Node noLoop(Node head1,Node head2){
        if(head1 == null || head2 == null){
            return null;
        }
        Node cur1 = head1;
        Node cur2 = head2;
        int n = 0;
        while (cur1.next != null){
            n++;
            cur1 = cur1.next;
        }
        while (cur2.next != null){
            n--;
            cur2 = cur2.next;
        }
        if(cur1 != cur2){
            return null;
        }
        cur1 = n > 0 ? head1 : head2; //找到长链表
        cur2 = cur1 == head1 ? head2 : head1; //找到短链表
        n = Math.abs(n);
        while (n != 0){
            n--;
            cur1 = cur1.next;
        }
        while (cur1 != cur2){
            cur1 = cur1.next;
            cur2 = cur2.next;
        }
        return cur1;
    }
    
	//loop1和loop2都有环
    public static Node bothLoop(Node head1,Node loop1,Node head2,Node loop2){
        Node cur1 = null;
        Node cur2 = null;
        //两个链表先相交再共同入环
        if (loop1 == loop2){
            cur1 = head1;
            cur2 = head2;
            int n = 0;
            while (cur1 != loop1){
                n++;
                cur1 = cur1.next;
            }
            while (cur2 != loop2){
                n--;
                cur2 = cur2.next;
            }
            cur1 = n > 0 ? head1 : head2;
            cur2 = cur1 == head1 ? head2 : head1;
            n = Math.abs(n);
            while(n != 0){
                cur1 = cur1.next;
            }
            while (cur1 != cur2){
                cur1 = cur1.next;
                cur2 = cur2.next;
            }
            return cur1;
        }else {
            //loop1继续走,如果在转回自己的过程中
            cur1 = loop1.next;
            while(cur1 != loop1){
                //遇到loop2,则有相交,则返回loop1或loop2都可以
                if(cur1 == loop2){
                    return loop1;
                }
                cur1 = cur1.next;
            }
            return null;
        }
    }

    //主函数
    public static Node getIntersectNode(Node head1,Node head2){
        if(head1 == null || head2 == null){
            return null;
        }
        Node loop1 = getLoopNode(head1);
        Node loop2 = getLoopNode(head2);
        if (loop1 == null || loop2 == null){
            return noLoop(head1, head2);
        }
        if (loop1 != null || loop2 != null){
            return bothLoop(head1, loop1, head2, loop2);
        }
        return null;
    }
}

            return true;

        Node r = null;

            head = queue.poll();
            l = head.left;
            r = head.right;

                queue.add(l);
            }
            if(r != null){
                queue.add(r);
            }
            //找到第一个左右孩子不双全的节点
            if(l == null || r == null){
                leaf = true;
            }
        }
        return true;
    }

       for(Node node : graph.nodes.values()){
           inMap.put(node, node.in);

           if (node.in == 0){
               zeroInQueue.add(node);
           }

       //结果
       List<Node> result = new ArrayList<>();

           //弹出入度为0的点,加到result中

           result.add(cur);
           //擦掉影响,把和cur有关的所有节点入度-1

               if(inMap.get(next) == 0){
                   zeroInQueue.add(next);
               }
           }
       }

37.kruskal算法（最小生成树）

适用范围：无向图

每次取权重最小的边，看是否生成环（开始时每个点放到一个集合里，每次加边的时候看新加入的节点和原来节点是否在一个集合中，没在一个集合就说明没有环，就把两个点合成一个集合）。生成环了就不要了。

public static class EdgeComparator implements Comparator<Edge>{
    @Override
    public int compare(Edge o1,Edge o2){
        return o1.weight - o2.weight;
    }
}

public static Set<Edge> kruskalMST(Graph graph){
    UnionFind unionFind = new UnionFind();
    unionFind.makeSets(graph.nodes.values());
    //通过比较器实现每次选择最小权重的边
    PriorityQueue<Edge> priorityQueue = new PriorityQueue<>(new EdgeComparator());
    for (Edge edge : graph.edges){	//M条边
        priorityQueue.add(edge);	// O(logM)
    }
	Set<Edge> result = new HashSet<>();
    while (!priorityQueue.isEmpty()){	//M条边
        Edge edge = priorityQueue.poll();	//O(logM)
        //每次弹出时判断from和to是不是在一个集合里
        if (!unionFind.isSameSet(edge,from, edge,to)) {	//O(1)
            //不在一起,把边加入集合,并把两个集合合并
            result,add(edge);
            unionFind.union(edge.from, edge.to);
        }
    }  
	return result;
}

38.prim算法（最小生成树）

适用范围：无向图

任选一个起始点，解锁他对应的边，选择权重最小的边，拽进对应的点，然后再解锁相应的边，再从所有被解锁的边中选择最小的。

public static class EdgeComparator implements Comparator<Edge>{
    @Override
    public int compare(Edge o1,Edge o2){
        return o1.weight - o2.weight;
    }
}

public static Set<Edge> primMST(Graph graph){
    //解锁的边进入小根堆
    PriorityQueue<Edge> priorityQueue = new PriorityQueue<>(new EdgeComparator());
    HashSet<Node> set = new HashSet<>();

    Set<Edge> result = new HashSet<>(); //依次挑选的边在result中

    for(Node node : graph.nodes.values()){  //随机挑点,node为开始点
        if(!set.contains(node)){
            set.add(node);
            for (Edge edge : node.edges){
                    priorityQueue.add(edge);    //选择node对应边中按照从小到大的顺序加入
            }
            while (!priorityQueue.isEmpty()){
                Edge edge = priorityQueue.poll(); //弹出解锁的边中最小的边
                Node toNode = edge.to; //边对应一些点
                if (!set.contains(toNode)){ //这个点是新的点
                    set.add(toNode);
                    result.add(edge);
                    for (Edge nextEdge : toNode.edges){
                        priorityQueue.add(nextEdge);
                    }
                }
            }
        }
    }
    return result;
}

39.Dijkstra算法（单元最短路径算法）

适用范围：可以有为负数的边，但是不能有累加和为负数的环

规定一个出发点，计算从这个点到其余所有点的距离最小

public static Node getMinDistanceAndUnselectedNode(HashMap<Node,Integer> distanceMap,HashSet<Node> torchedNodes){
    Node minNode = null;
    int minDistance = Integer.MAX_VALUE;
    for(Map.Entry<Node,Integer> entry : distanceMap.entrySet()){
        Node node = entry.getKey();
        int distance = entry.getValue();
        //如果距离比最小距离小,且这个点不是已经使用过的点,则让最小距离变成当前距离,并把node对应上
        if (!torchedNodes.contains(node) && distance <minDistance){
            minNode = node;
            minDistance = distance;
        }
    }
    return minNode;
}

public static HashMap<Node,Integer> diskstra(Node head){
    // 从head出发到所有点的最小距离
    // key:从head出发到达key
    // value:从head出发到达key的最小距离
    // 如果在表中,没有T的记录,含义是从head出发到T这个点的距离为正无穷
    HashMap<Node,Integer> distanceMap = new HashMap<>();
    //把当前节点到自己的距离为0加入
    distanceMap.put(head, 0);
    //已经求过距离的节点放入selectedNodes中,以后再也不碰
    HashSet<Node> selectedNodes = new HashSet<>();
    //从distanceMap中找到最小的记录,但是这个记录不能是selectedNodes中的
    Node minNode = getMinDistanceAndUnselectedNode(distanceMap,selectedNodes);
    //直到都在selectedNodes中,导致distanceMap无节点可选了,就停止循环
    while (minNode != null){
        int distance = distanceMap.get(minNode);    //原节点到当前minNode的距离
        for(Edge edge : minNode.edges){
            Node toNode = edge.to;
            //如果toNode(当前minNode连接的点)在distanceMap中没找到记录,则更新每个边和minNode连接的权重
            //没路加路
            if(!distanceMap.containsKey(toNode)){
                distanceMap.put(toNode, distance + edge.weight);
            }
            //新增后需要判断toNode的点之前的距离和现在的距离哪个小
            //有路更新
            distanceMap.put(edge.to, Math.min(distanceMap.get(toNode), distance + edge.weight));
        }
        //处理完当前节点,把他锁入selectedNodes
        selectedNodes.add(minNode);
        //再从没被锁的里面找到最小距离的点
        minNode = getMinDistanceAndUnselectedNode(distanceMap, selectedNodes);
    }
    return distanceMap;
}

public static class TrieNode {
    public int pass; //节点被走过几次
    public int end; //以该节点为字符串终点的次数
    public TrieNode[] nexts;    //走向每个字母的路
    //如果不是字母表示字符,可以创建HashMap<Char,Node> nexts;
    //如果希望下级的路是有序的方式,可以创建TreeMap<Char,Node> nexts;

    public TrieNode() {
        pass = 0;
        end = 0;
        nexts = new TrieNode[26];
    }
}

public static class Trie {
    //头节点
    private TrieNode root;

    public Trie() {
        root = new TrieNode();
    }

    //加入一个字符串
    public void insert(String word) {
        if (word == null) {
            return;
        }
        char[] chs = word.toCharArray();
        TrieNode node = root;
        node.pass++;
        int index = 0;
        for (int i = 0; i < chs.length; i++) { //遍历字符串
            index = chs[i] - 'a'; //判断走哪个字母的路
            //查询是否有走向当前字母的路,如果路不存在,则新建节点连上路
            if (node.nexts[index] == null) {
                node.nexts[index] = new TrieNode();
            }
            //否则走到字符串字母对应的节点,pass+
            node = node.nexts[index];
            node.pass++;
        }
        //把终止节点的end+
        node.end++;
    }

    //查询word这个单词之前加入了几次
    public int search(String word) {
        if (word == null) {
            return 0;
        }
        char[] chs = word.toCharArray();
        TrieNode node = root;
        int index = 0;
        for (int i = 0; i < chs.length; i++) {
            index = chs[i] - 'a';
            //如果这个边没有能找到的节点,则说明这个word没有被加入过
            if (node.nexts[index] == null) {
                return 0;
            }
            //如果有节点就走到头,结尾的end值就是word出现的次数
            node = node.nexts[index];
        }
        return node.end;
    }

    //所有加入的字符串中,有几个是以pre这个字符串作为前缀的
    public int prefixNumber(String pre) {
        if (pre == null) {
            return 0;
        }
        char[] chs = pre.toCharArray();
        TrieNode node = root;
        int index = 0;
        for (int i = 0; i < chs.length; i++) {
            index = chs[i] - 'a';
            if (node.nexts[index] == null) {
                return 0;
            }
            node = node.nexts[index];
        }
        return node.pass;
    }

    public void delete(String word){
        if(search(word) != 0){
            char[] chs = word.toCharArray();
            TrieNode node = root;
            node.pass--;
            int index = 0;
            for(int i = 0; i< chs.length;i++){
                index = chs[i] - 'a';
                //把沿途每个node对应的pass--
                if(--node.nexts[index].pass == 0){
                    //如果pass减到0说明这个节点没有用了,则把节点所有内容都释放掉
                    node.nexts[index] = null;
                    return;
                }
                node = node.nexts[index];
            }
            //把最后一个node的end--
            node.end--;
        }
    }
}

public static int findMaximizedCapital(int k,int W,int[] Profits,int[] Capital){
    PriorityQueue<Node> minCostQ = new PriorityQueue<>(new MinCostComparator());
    PriorityQueue<Node> maxProfitQ = new PriorityQueue<>(new MaxProfitComparator());
    // 所有项目扔到被锁池中，花费组织的小根堆
    for (int i = 0;i < Profits.length; i++){
        minCostQ.add(new Node(Profits[i],Capital[i]));
    }
	for (int i = 0 ;i < k;i++){ //进行K轮
        //能力所及的项目,全解锁
        while (!minCostQ.isEmpty() && minCostQ.peek().c <= W){
            maxProfitQ.add(minCostQ.poll());
        }
        if (maxProfitQ.isEmpty()){
            return W;
        }
        W += maxProfitQ.poll().p;
    }
    return W;
}

46.N皇后问题

N皇后问题是指在N*N的棋盘上要摆N个皇后，要求任何两个皇后不同行、不同列，也不在同一条斜线上。 给定一个整数n，返回n皇后的摆法有多少种。

n=1，返回1。 n=2或3，2皇后和3皇后问题无论怎么摆都不行，返回0。 n=8，返回92。

public class NQueens {
    public static int num1(int n) {
        if (n < 1) {
            return 0;
        }
        //每行的皇后放在了哪一列
        int[] record = new int[n];  //record[i]:第i行的皇后,放在了第几列
        return process1(0, record, n);
    }

    //record[0..i-1]的皇后一定不共行,不共列,不共斜线
    //i:目前来到了第几行,record[0..i-1]:之前的行放过哪些皇后,n:整体一共多少行,返回值:摆完合法的皇后,合理的摆法有多少种
    public static int process1(int i, int[] record, int n) {
        if (i == n) { //终止行
            return 1;   //说明找到了一种合法的
        }
        int res = 0;
        for (int j = 0; j < n; j++) { //当前行永远在i行,尝试i行所有的列 -> j
            if (isValid(record, i, j)) {    //检查是否冲突
                record[i] = j;  //如果没有冲突,则说明本行找到了,转到下一行接着找
                res += process1(i + 1, record, n);  //找遍所有行,如果有一个满足条件的则res++
            }
        }
        return res;
    }
    //检查record[0..i-1]
    public static boolean isValid(int[] record,int i, int j){
        for(int k = 0 ; k< i ; k++){    //之前k行的某个皇后
            //和之前的某个皇后:列共值 || 斜率45°(当前要放皇后的点列坐标绝对值和行坐标绝对值一致)
            if(j==record[k] || Math.abs(record[k] - j) == Math.abs(i-k)){
                return false;
            }
        }
        return true;
    }
}

public class NQueens {
    //不能超过32皇后问题,超过需要把int -> long
    public static int num2(int n) {
        if (n < 1 || n > 32) {
            return 0;
        }
        //申请一个后n位为1,前面都是0的数
        int limit = n == 32 ? -1 : (1 << n) - 1;
        return process2(limit,0, 0, 0);
    }
    //colLim:列限制,1的位置不能放皇后,0可以
    //leftDiaLim:左斜线限制,1的位置不能放皇后,0可以
    //rightDiaLim:右斜线限制,1的位置不能放皇后,0可以
    public static int process2(int limit, int colLim, int leftDiaLim, int rightDiaLim ){
        if(colLim == limit){        //相当于所有列都有皇后了
            return 1;
        }
        //所有可以填皇后的位置(pos上为1的表示可以放皇后的位置).或运算,找到总限制
        int pos = limit & (~(colLim | leftDiaLim | rightDiaLim));
        int mostRightOne = 0;
        int res = 0;
        while (pos != 0){
            mostRightOne = pos & (~pos + 1); //提取出候选皇后中最右侧的1
            pos = pos - mostRightOne; //把他从候选皇后中减掉
            //更新下一行的列限制与左右皇后限制
            res += process2(limit, colLim | mostRightOne , (leftDiaLim | mostRightOne) << 1,(rightDiaLim | mostRightOne) >>> 1);
        }
        return res;
    }
}

50.打印一个字符串的全部排列，要求不要出现重复的排列

对每一个字符，比如从i位置开始，则前面i-1位置已经固定了，则第i位置元素可以是后面还没固定的任意一个元素，则把他们分别换到i位置构成一种选择，每种选择都做了以后就可以去i+1位置继续全部递归了，最后需要把字符串回溯到原来的情况，否则字符串就变了

public static void process(char[] str,int i, ArrayList<String> res){
    if(i == str.length){
        res.add(String.valueOf(str));
    }
    //str[i..]范围上,所有的字符都可以在i位置上,后续去进行尝试
    //str[0..i-1]:之前做出的选择
    boolean[] visit = new boolean[26];	//visit[0]:表示'a'是否被试过
    for(int j = i;j<str.length;j++){
        if(!visit[str[j] - 'a']){	//分支限界:当前位置的字符没有被试过,才去试
            visit[str[j] - 'a'] = true;
        	swap(str,i,j);
        	process(str,i+1,res);
        	swap(str,i,j);	//回溯到之前的位置,否则字符串就变了
        }
    }
}

51.纸牌问题

给定一个整型数组arr，代表数值不同的纸牌排成一条线。玩家A和玩家B依次拿走每张纸牌，规定玩家A先拿，玩家B后拿，但是每个玩家每次只能拿走最左或最右的纸牌，玩家A和玩家B都绝顶聪明（会把最差的牌留给对方，他们决定自己拿到最差的牌）。请返回最后获胜者的分数。

举例：arr=[1,2,100,4]。开始时，玩家A只能拿走1或4。如果开始时玩家A拿走1，则排列变为[2,100,4]，接下来玩家B可以拿走2或4，然后继续轮到玩家A..如果开始时玩家A拿走4，则排列变为[1,2,100]，接下来玩家B可以拿走1或100，然后继续轮到玩家A... 玩家A作为绝顶聪明的人不会先拿4，因为拿4之后，玩家B将拿走100。所以玩家A会先拿1让排列变为[2，100,4]，接下来玩家B不管怎么选，100都会被玩家A拿走。玩家A会获胜分数为101。所以返回101。

arr=[1,100,2]。开始时，玩家A不管拿1还是2，玩家B作为绝顶聪明的人，都会把100拿走。玩家B会获胜分数为100。所以返回100。

public static int win(int[] arr) {
    if (arr == null || arr.length == 0) {
        return 0;
    }
    return Math.max(f(arr, 0, arr.length - 1), s(arr, 0, arr.length - 1));
}

public static int f(int[] arr, int i, int j) {
    if (i == j) {
        return arr[i];
    }
    return Math.max(arr[i] + s(arr, i + 1, j), arr[j] + s(arr, i, j - 1));
}

public static int s(int[] arr, int i, int j) {
    if (i == j) {
        return 0;
    }
    return Math.min(arr[i] + f(arr, i + 1, j), arr[j] + f(arr, i, j - 1));
}

        stack.push(result);

    }

53.字符串转化问题

规定1和A对应、2和B对应、3和C对应。

那么一个数字字符串比如"111”，就可以转化为"AAA"、"KA"和"AK"

给定一个只有数字字符组成的字符串str，返回有多少种转化结果。

思路：0~i-1：确定好了已经，找i位置以后有多少种方式。

  • i位置为0，不存在，直接return 0；
  • i位置为3-9，则不能和i+1位置结合，只有一种决定，只能将当前的字符做转换；
  • i位置为1，总可以和i+1结合，让i+2自由转换
  • i位置为2，i+1位置为0-6，可以结合，i+1位置为7-9，不能结合
public static int process(char[] chs,int i){
    if(i == chs.length){	//到达了最后一位(终止位置),将之前做的决定整体构成一种结果
        return 1;
    }
    if(chs[i] == '0'){
        return 0;
    }
    if(chs[i] == '1'){
        int res = process(chs,i+1); //i单独作为一部分,看后续有多少种
        if(i+1 < chs.length){
            res += process(chs,i+2); //i和i+1看作一部分,看i+2以后有多少种
        }
        return res;
    }
    if(chs[i] == '2'){
        int res = process(chs,i+1);	//i单独作为一部分,看后续有多少种
        if(i+1 < chs.length && (chs[i+1] >= '0' && chs[i+1] <= '6')){
            res += process(chs,i+2);
        }
        return res;
    }
    //chs[i] == '3~9'只能把当前位置做决定
    return process(chs,i+1);
}

        infect(m, i, j - 1, N, M);
        infect(m, i, j + 1, N, M);

59.LRU缓存（手写双向链表）

请你设计并实现一个满足 LRU (最近最少使用) 缓存 约束的数据结构。

实现 LRUCache 类：

  • LRUCache(int capacity) 以正整数作为容量 capacity 初始化 LRU 缓存
  • int get(int key) 如果关键字 key 存在于缓存中，则返回关键字的值，否则返回 -1 。
  • void put(int key, int value) 如果关键字 key 已经存在，则变更其数据值 value ；如果不存在，则向缓存中插入该组 key-value 。如果插入操作导致关键字数量超过 capacity ，则应该逐出最久未使用的关键字。
  • 函数 get 和 put 必须以 O(1) 的平均时间复杂度运行。
public class LRUCache {
    //创建一个双向链表
    class DLinkedNode{
        int key;
        int value;
        DLinkedNode prev;
        DLinkedNode next;
        public DLinkedNode() {}
        public DLinkedNode(int k,int v){
            key = k;
            value = v;
        }
    }
    //创建哈希表,用于存储缓存
    private Map<Integer,DLinkedNode> cache = new HashMap<>();
    private int size;   //判断当前长度是否超过capacity
    private int capacity;
    private DLinkedNode head,tail;  //伪头部和伪尾部

    public LRUCache(int capacity) {
        this.size = size;
        this.capacity = capacity;
        head = new DLinkedNode();
        tail = new DLinkedNode();
        head.next = tail;
        tail.prev = head;
    }
    
    public int get(int key) {
        //从缓存中查找
        DLinkedNode node = cache.get(key);
        if(node == null){
            return -1;
        }
        //找到了移到双向链表头(代表最近被使用)
        moveToHead(node);
        return node.value;
    }
    
    public void put(int key, int value) {
        //从缓存中找
        DLinkedNode node = cache.get(key);
        //没找到
        if(node == null){
            //加入双向链表
            DLinkedNode newNode = new DLinkedNode(key,value);
            //加入缓存
            cache.put(key,newNode);
            //加到双向链表头
            addToHead(newNode);
            ++size; //缓存数+1
            if(size > capacity){    //超过了最大缓存数
                DLinkedNode tail = removeTail();//删除尾部(最久未使用)
                cache.remove(tail.key);//在缓存中删掉最久未使用的
                --size;//缓存数-1
            }
        }else { //缓存中有
            node.value = value;  //更新链表中的值
            moveToHead(node);   //移到链表头
        }
    }

    private void addToHead(DLinkedNode node){
        node.prev = head;
        node.next = head.next;
        head.next.prev = node;
        head.next = node;
    }
    private DLinkedNode removeTail(){
        DLinkedNode res = tail.prev;
        removeNode(res);
        return res;
    }
    private void removeNode(DLinkedNode node){
        node.next.prev = node.prev;
        node.prev.next = node.next;
    }
    private void moveToHead(DLinkedNode node){
        removeNode(node);
        addToHead(node);
    }
}
